The Definition of a Derivative

The derivative of a function \(f(x)\) is defined as: $$ f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \tag{1} $$

Note that there are (confusingly) multiple notations for derivatives. However, each one has their own benefit. In Lagrange notation, a prime mark (\('\)) denotes a derivative. Leibniz notation shows the functional relationship between variables. Below are the first derivatives of a function \(f\) at \(x\).

  • Lagrange notation: \(f'(x)\)
  • Leibniz notation: \(\frac{df}{dx}(x)\) or \(\frac{df(x)}{dx}\) or \(\frac{d}{dx}f(x)\)

With that, let us look at some simple examples.

Find the derivative of \(f(x)\) using the definition of the derivative. $$f(x) = 2x$$
Show solution
The function \(f(x)\) is plugged into the definition of the derivative. With some simple algebra, $$\begin{align} f'(x) &= \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \\ &= \lim_{h \to 0} \frac{2(x + h) - 2x}{h} \\ &= \lim_{h \to 0} \frac{2x + 2h - 2x}{h} \\ &= \lim_{h \to 0} \frac{2h}{h} \\ &= \lim_{h \to 0} 2\\ &= 2 \end{align}$$ Therefore, the derivative is, $$f'(x) = 2$$
Find the derivative of \(V(x)\) using the definition of the derivative. $$V(x) = \sqrt{2x + 3}$$
First, place it in the definition of the derivative. $$\begin{align} V'(x) &= \lim_{h \to 0} \frac{V(x + h) - V(x)}{h} \\ &= \lim_{h \to 0} \frac{\sqrt{2(x + h) + 3} - \sqrt{2x + 3}}{h} \end{align}$$ In this case, when radicals are present in an equation, rationalization is used. To recall, rationalization is done by multiplying by its conjugate. In this case, the conjugate of \(\sqrt{2(x + h) + 3} - \sqrt{2x + 3}\) is \(\sqrt{2(x + h) + 3} + \sqrt{2x + 3}\). Thus, $$\begin{align} V'(x) &= \lim_{h \to 0} \frac{\sqrt{2(x + h) + 3} - \sqrt{2x + 3}}{h} \\ &= \lim_{h \to 0} \frac{\sqrt{2(x + h) + 3} - \sqrt{2x + 3}}{h} \left ( \frac{\sqrt{2(x + h) + 3} + \sqrt{2x + 3}} { \sqrt{2(x + h) + 3} + \sqrt{2x + 3} } \right ) \\ &= \lim_{h \to 0} \frac{2(x + h) + 3 - (2x + 3)}{h(\sqrt{2(x + h) + 3} + \sqrt{2x + 3})} \\ &= \lim_{h \to 0} \frac{2x + 2h + 3 - 2x - 3}{h(\sqrt{2(x + h) + 3} + \sqrt{2x + 3} )} \\ &= \lim_{h \to 0} \frac{2h}{h(\sqrt{2(x + h) + 3} + \sqrt{2x + 3} )} \end{align}$$ With this, factors of \(h\) remain on both the numerator and denominator which can be divided out. Afterwards, the limit is evaluated as usual. $$\begin{align} V'(x) &= \lim_{h \to 0} \frac{2}{\sqrt{2(x + h) + 3} + \sqrt{2x + 3}} \\ &= \frac{2}{\sqrt{2x + 3} + \sqrt{2x + 3}} \\ &= \frac{2}{2\sqrt{2x + 3}} \\ &= \frac{1}{\sqrt{2x + 3}} \end{align}$$ Thus, the derivative of \(V(x)\) is, $$V'(x) = \frac{1}{\sqrt{2x + 3}}$$
