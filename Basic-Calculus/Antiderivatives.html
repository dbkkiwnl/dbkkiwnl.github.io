Antiderivatives

Beginning
Rules of Integration â†’

In the previous chapter, the process of differentiation was defined. The inverse of addition is subtraction, and the inverse of multiplication is division. In this chapter, the inverse operation of differentiation will be explored.

Think of it as this: given a function f, is there a function F whose derivative is f?

A function F is an antiderivative of the function f on an interval I if F'(x) = f(x) for every value of x in I.

Observe the function f(x) = 6x2 + 2x. Its antiderivative is F(x) = 2x3 + x2. We can check by seeing that F'(x) = 6x2 + 2x, which is equal to the original function f(x).

The antiderivative F of any function f is not unique.

In fact, we can have any constant after 6x2 + 2x, which still would be an antiderivative. Thus, any function of the form F'(x) = 6x2 + 2x + C where C ∈ ℝ is an antiderivative.

If F is an antiderivative of f on an interval I, then every antiderivative of f on I is given by F(x) + C, where C is an arbitrary constant.

This means that there are an infinite number of antiderivatives for any given function.

Determine an antiderivative of the following functions.
(a) \(6x^2 + 2x\)
(b) \(24x^3 - 9x^2 - 1\)
(c) \(5x^4 + 3x^2\)

(a) \(2x^3+ x^2\)
(b) \(6x^4 - 3x^3 - x\)
(c) \(x^5 + x^3\)
Given a function f(x) and an antiderivative F(x), the set of all antiderivatives of f(x) is the indefinite integral of f, denoted by $$\int f(x)dx = F(x) + C \tag{1}$$

Terminology:

  • The symbol \(\int\), the integral sign, denotes the operation of antidifferentiation.
  • The function f is called the integrand.
  • The integral sign is always paired with dx, denoting the variable of integration. (dx, dt, dv...)
  • F(x) + C is the general antiderivative of f. Meanwhile, each antiderivative of f is called the particular antiderivative.
