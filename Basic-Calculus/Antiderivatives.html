Antiderivatives

Beginning
Antiderivatives of Algebraic Functions â†’

In the previous chapter, the process of differentiation was defined. The inverse of addition is subtraction, and the inverse of multiplication is division. In this chapter, the inverse operation of differentiation will be explored.

Think of it as this: given a function f, is there a function F whose derivative is f?

A function F is an antiderivative of the function f on an interval I if F'(x) = f(x) for every value of x in I.

Observe the function f(x) = 6x2 + 2x. Its antiderivative is F(x) = 2x3 + x2.

We can check by seeing that F'(x) = 6x2 + 2x, which is equal to the original function f(x).

The antiderivative F of any function f is not unique.

In fact, we can have any constant after 6x2 + 2x, which still would be an antiderivative. Thus, any function of the form \(F'(x) = 6x^2 + 2x + C\) where \(C\in\mathbb{R}\) is an antiderivative.

If F is an antiderivative of f on an interval I, then every antiderivative of f on I is given by F(x) + C, where C is an arbitrary constant.
